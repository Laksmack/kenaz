What this is

Futhark is a suite of small, focused macOS apps - each one genuinely useful on its own, and each one built to give Claude the same access you have. Not a dashboard or a wrapper, but real tools that happen to speak Claude's language.

Each app is named for an Elder Futhark rune whose meaning matches its purpose. They are designed to work together - Kenaz pulls CRM context from HubSpot while you draft, Dagaz surfaces tasks from Raidō when you plan your day.
